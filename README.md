# HTML_Test_App
RareTECH2025夏ハッカソン参加のテスト用  

# 課題について  
1. HTML_Test_Appというリポジトリをご自身のGitHub上に作成してください
2. ローカルにクローンしてください
3. HTMLとCSSを使って、ご自身の自己紹介ページを作成してください
4. 自己紹介の内容は、Mattermost名と、期の情報、趣味を表示してください
5. ご自身のGitHubにpushしてください
